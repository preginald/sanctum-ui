# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Light-mode surface tokens (`surface-light-low`, `surface-light-high`)
- `ErrorBoundary` component for graceful render error handling
- `CHANGELOG.md` and `CONTRIBUTING.md` documentation

## [0.1.0] - 2025-06-01

### Added

- React 19 support
- GitHub Packages publish workflow
- Chromatic visual regression CI

### Changed

- Upgraded peer dependency to React 18 or 19

## [0.0.0] - 2025-05-01

### Added

- Initial release of `@preginald/sanctum-ui`
- Tailwind CSS preset with Digital Sanctum design tokens
- Button, Input, StatusBadge, Card, Alert, and Toast components
- Vitest + Testing Library + axe-core test suite
- Storybook for component development
- ESLint flat config with TypeScript support
